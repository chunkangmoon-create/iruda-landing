# Kixx BI 가이드 - Basic System

## 컬러 팔레트

### Kixx 주요 색상

#### Primary Color
- **Deep Orange** (슬로건 색상)
  - Pantone 173C
  - CMYK: 0 / 69 / 100 / 4
  - RGB: 232 / 108 / 30
  - HEX: `#E86C1E`

- **Black** (워드마크 기본색)
  - Pantone 173C
  - CMYK: 0 / 69 / 100 / 4
  - RGB: 232 / 108 / 30
  - HEX: `#E86C1E`

#### Secondary Color
- **Light Gray**
  - Pantone 420C
  - CMYK: 0 / 0 / 0 / 15
  - RGB: 230 / 230 / 230
  - HEX: `#E6E6E6`

- **Gray**
  - Pantone 421C
  - CMYK: 0 / 0 / 0 / 30
  - RGB: 202 / 202 / 202
  - HEX: `#CACACA`

- **Jade Green** (보조색)
  - Pantone 321C
  - CMYK: 100 / 10 / 45 / 5
  - RGB: 0 / 153 / 153
  - HEX: `#009999`

### 배경색 우선순위
White > Gray > Jade Green > Black

## 타이포그래피

### 기본 폰트
- **Noto Sans** (영문 및 외국어)
  - Regular: 본문
  - Bold: 제목

### 폰트 다운로드
https://www.google.com/get/noto/

## 로고 사용 규칙

### 최소 크기
- **인쇄**: 5mm 이상
- **웹**: 15px 이상

### 여백 (Isolation Area)
- 로고 주변에 일관된 보호 공간 필요
- 기본 여백: X(10x) × 5x

### 금지 사항
❌ 색상 변경 또는 추가
❌ 복잡한 패턴 위에 배치
❌ 그림자 효과 추가
❌ 비율 변경
❌ 기울임 처리
❌ 아웃라인 추가
❌ 로고 재생성
❌ 왜곡

## 슬로건

### "All Ways with You"
- **색상**: Deep Orange (#E86C1E)
- **스타일**: Italic
- **최소 크기**
  - 인쇄: 2mm 이상
  - 웹: 5px 이상

## 설계 원칙

### 색상 사용
- 그라데이션 표현이 기본 (필요시 단색, 마지막 수단으로 B/W)
- 모든 배경색에서 명확하게 표현되어야 함

### 로고 배치
- 일관된 아이덴티티 표현
- 매체 특성에 맞는 형태 선택 (가로/세로)
- 비례 규정 준수

### 공간 규정
- Isolation area를 반드시 확보
- 충분한 최소 공간 유지
- 다른 요소와의 분리 필수

## 웹 적용 시 중요 사항

1. **색상 정확도**: HEX 값 정확히 적용
2. **최소 크기 준수**: 15px 이하 사용 금지
3. **배경 명도**: 배경이 40% 이상 명도일 때 컬러 로고 사용
4. **여백 관리**: 로고 주변에 최소한의 여백 확보
5. **폰트**: Noto Sans 또는 시스템 폰트 조합 사용

## 랜딩페이지 디자인 적용

### 추천 색상 조합
- **주배경**: White / Light Gray
- **강조색**: Deep Orange (#E86C1E)
- **보조색**: Jade Green (#009999)
- **텍스트**: Black / Dark Gray

### 여백 전략
- 로고는 항상 충분한 여백 확보
- 섹션 간 구분에 컬러 활용
- 계층 구조 명확히

### 금지 패턴
- 복잡한 배경 위의 로고 배치
- 여러 색상 추가 또는 변경
- 로고 비율 변경
