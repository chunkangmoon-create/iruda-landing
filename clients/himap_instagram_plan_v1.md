# 하이맵의원 × 이루다 마케팅 — 인스타그램 실행 계획서

작성일: 2026-03-11
작성: 이루다 마케팅 에이전시

---

## 📌 이 문서를 읽기 전에

이 문서는 "인스타그램에 어떤 게시물을 올릴 것인가"에 대한 답이 아닙니다.

"환자가 하이맵의원을 어떻게 선택하는가"를 먼저 이해한 뒤, 그 행동 구조에 맞게 인스타그램을 어떻게 활용할 것인가를 설계한 문서입니다.

> 콘텐츠는 전략의 결과입니다. 전략 없는 콘텐츠는 조회수를 만들지 못하고, 조회수가 있어도 예약으로 이어지지 않습니다.

---

## 1. 환자는 어떻게 병원을 선택하는가

### 실제 유입 구조

하이맵의원 신환의 주요 유입 경로는 다음 순서입니다.

1. 네이버 검색
2. 유튜브
3. 지인 소개

이 구조에서 가장 중요한 사실이 하나 있습니다.

**환자는 병원을 먼저 찾지 않습니다.**

환자의 실제 행동 경로는 이렇습니다.

1. 증상 경험
2. 증상 검색 ("이명 왜 생기나", "잠 못 자는 이유")
3. 원인 탐색 ("이명 뇌 관련", "불면 기능의학")
4. 병원 검색 ("하이맵의원", "교대역 기능의학병원")
5. 병원 비교 (블로그, 플레이스 후기 확인)
6. 예약 및 방문

환자는 자신의 증상을 먼저 검색하고, 병원을 나중에 찾습니다.

### 인스타그램이 이 구조에서 할 수 있는 일

인스타그램은 예약을 직접 만드는 채널이 아닙니다. 인스타그램이 영향을 줄 수 있는 구간은 **증상 경험 → 증상 검색** 단계입니다.

즉, 인스타그램은 **퍼널의 시작을 만드는 채널**입니다. 환자가 자신의 문제를 인식하고, 검색을 시작하도록 만드는 것이 인스타그램의 역할입니다.

## 2. 전략 구조 — 2개의 꼭지

이루다 마케팅은 인스타그램 운영을 2개의 목표 축으로 나눠 운영합니다.

### 꼭지 1. 브랜드 검색 유도

> 네이버에서 "하이맵의원" 검색량을 늘린다

- 수단: 유기적 콘텐츠 (릴스, 카드뉴스)
- 성과 지표: 네이버 브랜드 검색량 증가, 릴스 도달 수, 저장 수

### 꼭지 2. 직접 전환

> 랜딩페이지 유입 → 신환 문의 전화 전환

- 수단: 메타 유료 광고
- 성과 지표: 랜딩페이지 클릭 수, 신환 문의 전화 수

---

## 3. 꼭지 1 — 브랜드 검색 유도

### 목표

인스타그램 콘텐츠를 보고 네이버에서 "하이맵의원"을 검색하게 만든다.

### 왜 검색이 목표인가

한국 의료 소비자의 행동 특성상, 환자는 인스타그램 광고를 보고 바로 예약하지 않습니다. 반드시 네이버에서 병원을 검색하고 블로그 후기, 플레이스 리뷰, 원장님 이력을 확인한 뒤 예약합니다.

따라서 인스타그램이 만들어야 하는 것은 "예약"이 아니라 **검색 행동**입니다.

### 핵심 메커니즘 — 문제 프레임 전환

이 꼭지의 핵심은 **환자의 문제 해석을 바꾸는 것**입니다.

- Before: "이명이 있다" → 이비인후과 검색
- After: "이명이 뇌 신경 문제일 수 있다" → 하이맵의원 검색

문제 해석이 바뀌면 검색어가 바뀌고, 병원 선택이 달라집니다.

---

### 콘텐츠 유형 3가지

#### 유형 1. 증상 공감 콘텐츠

**목적:** "내 이야기다" 반응을 만들어 알고리즘 확산 유도

**형식:** 릴스 또는 카드뉴스

**톤:** 설명이 아닌 공감. 환자의 경험을 그대로 묘사.

**예시 주제**

- 이명이 있는데 잠도 같이 깨나요?
- 검사 결과 정상인데 왜 이렇게 피곤하죠?
- 브레인포그? 머리가 안개 낀 것처럼 멍한 그 느낌
- 소화가 안 되는데 내시경은 깨끗하다고 했다면
- 스트레스받으면 꼭 이명이 심해지는 이유

**구조 공식**

1. **Hook** — 환자가 경험한 증상을 정확히 묘사 (1~2줄)
2. **공감** — "이런 경험 있으신가요?" 질문 또는 체크리스트
3. **반전** — "사실 이건 ○○의 신호일 수 있습니다"
4. **열린 결말** — "이유가 궁금하다면 → 하이맵의원 검색"

---

#### 유형 2. 원인 설명 콘텐츠

**목적:** 환자가 자신의 증상을 다른 관점으로 이해하게 만들기

**형식:** 릴스 (60~90초) 또는 슬라이드 카드

**톤:** 의학적 근거는 있지만 어렵지 않게. 원장님 직접 등장 또는 자막 구성.

**예시 주제**

- 이명은 귀 문제가 아닐 수 있습니다 (뇌 신경 관점)
- 불면증 약을 먹어도 잠 못 자는 이유
- 아토피가 피부과에서 낫지 않는 진짜 이유
- 기능의학 검사 vs 건강검진, 뭐가 다른가요?
- 이 5가지 증상이 같은 뿌리에서 나온다면?

**구조 공식**

1. **기존 인식** — 환자가 지금까지 생각하던 방식
2. **반전 제시** — "하지만 실제로는 이런 이유입니다"
3. **근거** — 데이터 또는 임상 경험 기반 설명 (7만 건, 22년)
4. **행동 유도** — "정확한 원인이 궁금하다면 → 하이맵의원 검색"

---

#### 유형 3. 신뢰 구축 콘텐츠

**목적:** 이미 검색을 시작한 환자에게 하이맵의원을 선택하게 만들기

**형식:** 치료 사례, 원장님 인터뷰, 숫자 기반 실적 카드

**예시 주제**

- 6년 이명 → 12주 만에 호전된 실제 사례
- 하이맵의원 기능의학 검사, 이렇게 진행됩니다
- "이 병원이 다른 이유" — 원장님 직접 설명
- 7만 건 데이터로 본 이명의 가장 흔한 원인 3가지
- 전국에서 오는 이유 — 제주도, 해외 교포까지

---

### 콘텐츠 운영 방식

#### 월별 집중 증상 테마

한 달에 여러 증상을 섞지 않고, 하나의 증상 클러스터를 집중합니다. 알고리즘 학습 속도가 빨라지고, 특정 분야 전문병원으로 인식됩니다.

| 월 | 집중 테마 |
| --- | --- |
| 1월 | 이명 + 불면 |
| 2월 | 장 문제 + 브레인포그 + 피부 |
| 3월 | 만성피로 + 부신피로 + 호르몬 |
| 4월 | 공황 + 불안 + ADHD |
| 5월 | 아토피 + 결절성양진 + 면역 |
| 6월 | 기능의학 검사 / 종합 건강관리 |

#### 월 8개 게시물 구성 (예시)

| 형식 | 수량 | 세부 내용 |
| --- | --- | --- |
| 릴스 (영상) | 4개 | 증상 공감 2개 + 원인 설명 2개 |
| 카드뉴스 | 3개 | 원인 설명 1개 + 신뢰 구축 2개 |
| 스토리 / 하이라이트 | 1개 | 검사 안내 또는 이벤트 |

#### CTA (행동 유도 문구) 원칙

모든 콘텐츠의 마지막은 아래 중 하나로 끝납니다.

- ✅ "네이버에서 하이맵의원 검색해보세요"
- ✅ "하이맵의원에서 원인 찾아보세요"
- ✅ "프로필 링크에서 더 알아보세요"
- ❌ "지금 예약하세요" — 사용하지 않음 (차가운 타겟에게 역효과)
- ❌ "링크 클릭하세요" — 사용하지 않음

### KPI — 꼭지 1

| 네이버 브랜드 검색량 | 네이버 서치어드바이저 | 월간 증가 추이 |

## 4. 꼭지 2 — 직접 전환 (메타 광고)

### 목표

메타 광고 → 기능의학 검사 랜딩페이지 → 신환 문의 전화 전환

### 전환 구조

메타 광고 소재 노출 → 랜딩페이지 클릭 → 전화 문의 (02-6232-7582) → 신환 예약

### 광고 소재 전략

#### 소재 A — 범용 (v1 랜딩 연결)

**타겟:** 기능의학, 건강관리, 만성 증상 관심사

**후크 문구 A/B 테스트 후보**

- "검사는 다 정상인데 왜 이렇게 아프죠?"
- "병원을 전전했는데 원인을 못 찾으셨나요?"
- "20년 경험, 7만 건 데이터. 당신의 증상 원인 찾아드립니다"

**연결 랜딩:** v1 — 기능의학 검사 전반

---

#### 소재 B — 이명 특화 (v2 랜딩 연결)

**타겟:** 이명, 난청 관심사 / 이명 콘텐츠 반응자 리타겟팅

**후크 문구 A/B 테스트 후보**

- "이비인후과에서 이상 없다는데, 왜 소리가 계속 들리죠?"
- "이명 6년, 12주 만에 소리가 사라졌습니다"
- "매미 소리, 윙 소리… 혼자 듣고 계신가요?"

**연결 랜딩:** v2 — 이명 특화

### 소재 C — 파트너십 애즈 (Partnership Ads)

> 요즘 메타에서 가장 성과가 좋은 광고 형식입니다. 소재 A·B와 병행 운영합니다.

#### 파트너십 애즈란?

파트너십 애즈는 브랜드(하이맵의원)가 크리에이터·인플루언서의 계정에서 직접 광고를 집행하는 메타의 공식 광고 형식입니다. 게시물 상단에 **"[크리에이터 계정]의 유료 파트너십"** 라벨이 붙으며, 크리에이터의 팔로워와 브랜드의 타겟 오디언스 모두에게 동시에 노출됩니다.

기존 브랜드 광고와의 가장 큰 차이는 **"병원이 하는 말"이 아니라 "사람이 하는 말"처럼 보인다는 것**입니다. 의료 콘텐츠에서 이 차이는 신뢰도와 클릭률에 직접적인 영향을 줍니다.

#### 파트너십 애즈의 특징

| 구분 | 일반 브랜드 광고 | 파트너십 애즈 |
| --- | --- | --- |
| 발신자 | 병원 공식 계정 | 크리에이터 개인 계정 |
| 신뢰도 | 광고처럼 보임 | 실제 경험담처럼 보임 |
| 피드 자연스러움 | 낮음 | 높음 (유기적 콘텐츠와 동일) |
| 댓글·반응 | 낮음 | 높음 |
| CTR (클릭률) | 낮음 | 일반 광고 대비 평균 2~3배 |

#### 하이맵의원에 적합한 이유

- 의료 콘텐츠는 **"누가 말하는가"가 메시지보다 중요**합니다. 병원 계정이 직접 "우리 병원 와보세요"라고 말하는 것보다, 실제 증상을 겪은 사람처럼 보이는 계정에서 "이 병원에서 원인을 찾았다"는 콘텐츠가 훨씬 신뢰를 만듭니다.
- 이명, 기능의학, 만성 증상 관련 크리에이터·환자 체험 계정과 협업 시 **해당 팔로워 전체가 잠재 타겟**이 됩니다.
- 크리에이터 콘텐츠는 광고 피로도가 낮아 **노출 대비 반응율(engagement rate)이 높습니다.**

#### 운영 방식

1. **협업 크리에이터 선정** — 건강·의료·기능의학 관련 인스타그램 계정 (팔로워 1만~10만 구간 마이크로 인플루언서 우선)
2. **콘텐츠 방향 기획** — 이루다 마케팅이 기획, 크리에이터가 자신의 말투로 제작
3. **파트너십 승인** — 메타 크리에이터 스튜디오에서 브랜드 승인 처리
4. **광고 집행** — 크리에이터 계정에서 하이맵의원 랜딩페이지로 연결
5. **성과 측정** — 일반 소재 A·B와 CTR, 전환율 비교

#### 콘텐츠 예시 방향

- 기능의학 검사를 실제로 받아본 경험 공유 ("20만원 기능의학 검사, 솔직 후기")
- 이명·불면 증상을 가진 크리에이터의 병원 방문기
- 건강·영양 전문 계정에서 "기능의학 검사가 건강검진과 다른 이유" 설명

> 💡 파트너십 애즈는 크리에이터 섭외 비용이 발생하지만, 일반 광고 대비 CPC(클릭당 비용)가 낮아 전체 광고 효율이 올라가는 경우가 많습니다. 초기에는 소규모로 테스트 후 성과에 따라 확대합니다.

## 5. 두 꼭지의 연결 구조

꼭지 1과 꼭지 2는 독립적으로 보이지만 실제로는 하나의 퍼널로 연결됩니다.

**꼭지 1 → 꼭지 2 시너지 흐름**

- 유기적 콘텐츠로 병원 인지 → 메타 광고 재노출 시 전환율 상승
- 메타 광고로 랜딩 유입 → 인스타 콘텐츠 발견 시 신뢰도 상승

꼭지 1에서 콘텐츠를 본 환자가 꼭지 2의 광고를 다시 보면 전환율이 올라갑니다. 꼭지 2 광고를 본 환자가 꼭지 1의 유기적 콘텐츠를 발견하면 신뢰도가 올라갑니다.
